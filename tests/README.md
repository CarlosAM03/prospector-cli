
# Tests

This directory contains auxiliary scripts used during the development of Prospector CLI.

These scripts are not part of the production pipeline.

Their purpose is to validate assumptions, benchmark performance, inspect DOM behavior and prototype extraction strategies before they are incorporated into the engine.

## Structure

```text

tests/

research/
integration/
unit/
performance/

```

## Notes

Research scripts document architectural decisions and may become obsolete once their findings have been incorporated into the project.
